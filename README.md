Tasques - Proyecto de Programación Estática y Laboratorio Web <br>

Este repositorio contiene el proyecto final desarrollado para la asignatura **Programación Estática y Laboratorio Web (2025)** en la Facultad de Informática de la Universidad Nacional del Comahue.

> Descripción

**Tasques** es una aplicación web diseñada para la gestión de tareas, enfocada en aplicar los pilares fundamentales del desarrollo frontend: estructura, estilo y funcionalidad dinámica en el lado del cliente.

> Estructura del Repositorio

El núcleo del proyecto se encuentra en la carpeta `tasques_V4_Otermin/` y está compuesto por:

* **`index.html`**: La estructura semántica del sitio.
* **Estilos (CSS)**: Definición de la interfaz visual, diseño responsivo y estética de la aplicación.
* **Lógica (JavaScript)**: Manejo de eventos, manipulación del DOM y gestión del estado de las tareas (agregar, completar o eliminar).
* **Recursos**: Imágenes u otros activos utilizados en la interfaz.

> Tecnologías Utilizadas

* **HTML5**: Estructura del contenido.
* **CSS3**: Estilos y maquetación.
* **JavaScript (Vanilla)**: Lógica de la aplicación y dinamismo.

> Visualización y Uso

Al ser un proyecto de programación estática:

1. **En local**: El proyecto se puede visualizar simplemente abriendo el archivo `index.html` en cualquier navegador moderno.
2. **Desde VS Code**: Se recomienda utilizar la extensión **Live Server** para una previsualización en tiempo real durante el desarrollo.
3. **Interactividad**: El usuario puede interactuar con la aplicación directamente desde la ventana del navegador para gestionar sus tareas.
